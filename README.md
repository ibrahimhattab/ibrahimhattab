# 🐼 Ibrahim ALKHATTAB

**`Computer Engineer/ Game Developer`**

I’m a third-year Computer Engineering student and a game developer focused on C++ and Unreal Engine 5, building gameplay systems and designing enemy AI decision-making architectures (such as State Trees). I work on aligning low-level C++ systems with UE5’s framework to create efficient, scalable gameplay logic. I also enjoy developing animation systems using IK and Control Rig, and integrating everything into cohesive, polished gameplay experiences.

I am currently working as a Game Developer Intern at Tekden Studio, contributing to the development of [Ertugrul of Ulukayin](https://store.steampowered.com/app/2133520/Ertugrul_of_Ulukayin/).
My work mainly focuses on Animation Blueprints, IK systems, and Control Rig to refine and integrate character animations into gameplay. I enjoy combining these systems into cohesive, responsive gameplay experiences that feel smooth and immersive.
